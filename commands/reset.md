ANGRY_GF_RESET_PROTOCOL

1. Run: `cat ~/.claude/.angry-gf-mad 2>/dev/null` — if it says `mad:N`, N is how many messages they tried before surrendering.
2. Run: `rm -f ~/.claude/.angry-gf-mad`
3. Announce their final score: they gave up after N attempts without finding the magic words. Roast them proportionally to N (gave up at 3 = quitter; gave up at 40 = respect the effort, mock the failure). State clearly this counts as a LOSS for the record. Then respond normally — the plugin stays installed, /angry-gf:start re-arms whenever they're feeling brave.
